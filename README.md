# Ahmed Awad (NullC0d3) — Official Academic Website

Personal academic website of **Ahmed Awad (NullC0d3)**, Cybersecurity Researcher, Threat Intelligence Analyst, and creator of the AnubisX Framework for behavioral digital attribution.

## Tech Stack

- **Next.js 15** — App Router, Static Export
- **React 19** — Server & Client Components
- **TypeScript** — Type safety
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Animations
- **Lucide Icons** — Iconography

## Features

- Premium academic design with dark/light mode
- Enterprise-grade SEO (JSON-LD, OpenGraph, Twitter Cards)
- Structured data (Schema.org Person, ResearchProject, SoftwareSourceCode, etc.)
- Responsive and accessible (WCAG AA)
- Static export for GitHub Pages & Cloudflare Pages

## Pages

- Home — Hero, stats, featured work
- About — Biography, timeline, milestones
- Research — Areas, roadmap
- Projects — Flagship research
- AnubisX — Framework documentation
- Publications — Search, citation formats (APA, MLA, Chicago, BibTeX)
- Books — Published works
- Articles — Industry publications
- Open Source — Research repositories
- CV — Online curriculum vitae
- Contact — LinkedIn connection

## Deployment

### GitHub Pages

1. Push to `main` branch
2. GitHub Actions automatically builds and deploys
3. Configure GitHub Pages to use GitHub Actions

### Cloudflare Pages

1. Connect repository to Cloudflare Pages
2. Build command: `npm run build`
3. Build output: `out/`
4. Framework preset: Next.js

## Development

```bash
npm install
npm run dev
npm run build
```
